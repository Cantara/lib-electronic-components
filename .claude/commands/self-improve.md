Analyze feedback from audited projects and propose improvements to templates and prompts.

Read and follow the prompt at `new-project-setup/prompts/self-improve.md`.

Key steps:
1. Read all project reports in `new-project-setup/feedback/project-reports/`
2. Read meta-learnings from `new-project-setup/feedback/meta-learnings.md`
3. Read current templates, prompts, and rubric
4. Analyze patterns: consistent modifications, low-scoring dimensions, missing content
5. Cross-reference with `new-project-setup/reference/industry-practices.md`
6. Generate prioritized improvement proposals with evidence and rationale
7. Append analysis summary to `new-project-setup/feedback/meta-learnings.md`
8. Present proposals to user for accept/reject/modify decisions
9. Apply accepted changes and update EVOLUTION.md

Note: All proposed changes require human review. This command proposes, it does not auto-apply.
